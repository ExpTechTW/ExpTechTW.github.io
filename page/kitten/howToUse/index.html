EWW(Earthquake Early Warning)及地震報告

kitten可在地震來臨時，提供地震警報；還可在地震來臨後，提供完整的地震報告。
+EEW here - 設定此頻道為EEW推播頻道
+EEW tag < 身分組 ID> - 設定EEW標註的身份組
+EWW add < 範例:台南市歸仁區> - 新增EWW倒計時地區
+EWW remove < 範例:台南市歸仁區> - 移除EWW倒計時地區
+report here - 將此頻道設為地震報告推播頻道
+report tag < 身分組 ID> - 設定地震報告標註的身份組
+EWW test - 發送EWW地震速報測試訊息(必須擁有權限3才可使用)

urlChecker

+urlChecker false - 關閉連結檢查功能
